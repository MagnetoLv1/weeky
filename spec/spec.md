## 1. 앱 한 줄 설명

주간 시간표를 시각적으로 관리할 수 있는 모바일 앱.
빈 칸 클릭으로 일정 추가, 드래그로 이동/크기 조정 가능.

## 4. 화면별 기능 정의

### 메인 시간표 화면

**보여줄 것**

- 상단: 요일 헤더 (기본 월~금, 설정에서 주말 표시 ON 시 월~일), 오늘 날짜 강조 표시 (파란 원)
- 좌측: 시간 타임라인 (세로, HH:MM 형식)
- 중앙: 7열 그리드 (요일별 일정 블록 표시)
- 일정 블록에 알림이 설정된 경우 🔔 아이콘 표시
- 우상단: 설정 아이콘 (⚙️)
- 상단 제목: "시간표"

**사용자 액션**

- 빈 셀 클릭 → 일정 추가
- 일정 블록 클릭 → 일정 편집
- 일정 블록 드래그 → 시간/요일 이동
- 일정 블록 상하단 드래그 → 시간 길이 조정 (10분 단위)
- 프린트 버튼 → 현재 주간 시간표 전체 인쇄
- 공유 버튼 → 주간 시간표 전체를 이미지로 변환 후 공유 시트 열기

**초기 상태**

- 앱 최초 실행 시 기본 시간표("시간표")가 자동 생성됨
- 초기 화면은 주간 시간표 화면 (MainScreen)

**빈 상태 (Empty State)**

- 일정이 없을 때 그리드만 표시 (별도 안내 문구 없음)
- 최소 단위: 10분
- 셀 높이: 10분 = 일정 높이 단위
- 시간 범위: 사용자가 설정에서 직접 지정 (시작 시간 ~ 종료 시간)
- 일정 블록 색상: 항목마다 다른 색상 지정 가능

### 일정 추가/편집 화면

**입력 항목**

- 제목 (필수)
- 부제/장소 (선택, 예: "302호")
- 시작 시간 / 종료 시간 (바텀시트 휠 피커, 10분 단위)
- 반복 요일 선택 (복수 선택 가능, 매주 반복)
- 색상 선택 (파스텔 팔레트 10종에서 선택)
- 메모/노트 (선택)
- 알림 설정 (선택): 토글 + 5/10/15/30분 전 선택

**액션**

- 저장 → 시간표에 블록 생성, 알림 설정 시 로컬 알림 등록
- 취소 → 이전 화면으로
- 편집 모드 진입 시 삭제 버튼 노출

### 멀티 시간표

**보여줄 것**

- 상단에 시간표 탭 또는 스와이프로 전환 가능
- 현재 활성화된 시간표 이름 표시
- 시간표 추가(+) 버튼
- 시간표가 2개 이상일 때만 삭제 버튼 노출 (1개일 때는 숨김)

**사용자 액션**

- 헤더 타이틀 영역 좌우 스와이프 → 전체 그리드 슬라이드 애니메이션과 함께 전환 (탭 UI 없음)
- 헤더에 ‹ 이름 › 표시 + 도트 페이지 인디케이터로 현재 위치 표시
- 시간표 추가(+) 버튼 (헤더 우측) → 새 빈 주간 시간표 생성, 슬라이드로 이동
- 시간표 탭을 길게 누르기 → 편집 모드 진입 (이름 변경 / 순서 드래그 변경)
- 시간표 삭제 → 경고 알림 표시 ("삭제하면 모든 일정이 사라집니다. 계속하시겠습니까?") → 확인 시 삭제
- [x] 설정 화면은 현재 보고 있는 시간표 기준 (앱 전역 설정 아님)
- [x] 설정 화면에서 현재 시간표 이름 변경(✏️), 시간 범위, 주말 표시 설정 가능
- [x] 시간표 삭제: 2개 이상일 때만 노출, Alert 확인 후 삭제
- [x] 각 시간표는 독립적인 timeRangeStart/End, showWeekends 보유

**전환 애니메이션**

- 좌우 슬라이드 전환 (카드 넘기는 느낌)
- 애니메이션 duration: 약 300ms

**규칙**

- 시간표 최소 1개 유지 필수
- 시간표 1개일 때 삭제 버튼 비노출
- 각 시간표는 독립적인 일정 데이터 보유

### 화면 줌 기능

**동작**

- 빈 공간 더블 탭 → 해당 요일 기준으로 줌인
- 줌인 상태: 선택한 요일 56%, 좌우 인접 요일 각 22%(약 30% 노출), 나머지 요일 숨김
- 줌인 상태에서 가능한 조작: 일정 클릭/편집, 빈 공간 롱프레스로 일정 추가
- 줌아웃: 빈 공간 더블 탭 또는 헤더 "전체" 버튼으로 복귀
- 줌 전환 애니메이션: 컬럼 너비 부드러운 확대/축소 (250ms)

**일정 추가 (빈 공간)**

- 롱프레스 (400ms) → 햅틱 진동 → 일정 추가 폼 열림 (해당 요일/시간 자동 설정)
- 더블탭 → 줌 토글

**이미지 공유 기능**

- 주간 시간표 전체를 PNG 이미지로 캡처
- 기기 기본 공유 시트(Share Sheet) 호출 → 카카오톡, 인스타, 사진 저장 등 가능
- 공유 이미지에 앱 워터마크 표시 여부: 미정 🔄

## 5. 데이터 구조

```
Timetable {
  id: string
  name: string           // 시간표 이름 (예: "학교", "학원")
  order: number          // 탭 순서
  schedules: Schedule[]  // 해당 시간표의 일정 목록
}

Schedule {
  id: string
  title: string          // 일정 이름 (예: "수학")
  subTitle: string?      // 부제/장소 (예: "302호")
  memo: string?          // 메모/노트
  dayOfWeek: number[]    // 0=월 ~ 6=일 (반복이므로 배열)
  startTime: string      // "09:00"
  endTime: string        // "10:30"
  color: string          // 블록 색상 (hex)
  isRepeating: boolean   // 매주 반복 여부
  notification: {
    enabled: boolean     // 알림 사용 여부
    minutesBefore: 5 | 10 | 15 | 30  // 고정 선택지
  }?
}

Settings {
  timeRangeStart: string  // 시간표 시작 시간 (예: "07:00")
  timeRangeEnd: string    // 시간표 종료 시간 (예: "23:00")
  showWeekends: boolean   // 토/일 표시 여부 (기본: false → 월~금만 표시)
  timeRangeEnd: string    // 시간표 종료 시간 (예: "23:00")
}
```

## 6. 핵심 동작 흐름

**일정 추가**

1. 시간표 빈 공간 탭
2. 일정 추가 화면 열림
3. 제목, 시간, 색상 입력
4. 저장 → 해당 위치에 블록 생성

**일정 이동 (드래그)**

1. 일정 블록 길게 누르기 (또는 즉시 드래그)
2. 원하는 요일/시간으로 드래그
3. 10분 단위로 스냅되며 이동
4. 손 떼면 해당 위치에 고정

**시간 길이 조정**

1. 블록 상단 또는 하단 핸들 드래그
2. 10분 단위로 스냅
3. 손 떼면 확정

## 8. 제약 조건

- 알림 시간 선택: 5분 전 / 10분 전 / 15분 전 / 30분 전 (고정 선택지)
- 반복 일정: 매주 반복 지원 (등록 시 요일 복수 선택 가능)
- 중복 일정: 허용 (동일 시간대 겹칠 경우 위에 덮어서 표시)
- 색상 선택: 파스텔 팔레트에서 선택 (자유 커스텀 없음)

## 9. Out of Scope (이번 버전 미포함 기능)

- 다국어 지원
