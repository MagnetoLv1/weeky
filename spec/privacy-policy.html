Weeky 개인정보처리방침

최종 업데이트: 2026년 3월 25일

Weeky(이하 "앱")는 사용자의 개인정보를 소중히 여깁니다. 본 방침은 앱이 어떤 정보를 수집하고 어떻게 사용하는지 설명합니다.

1. 수집하는 정보

Weeky는 어떠한 개인정보도 수집하지 않습니다. 사용자가 입력한 시간표 데이터(일정 이름, 시간, 색상 등)는 오직 기기 내부 저장소에만 저장되며, 외부 서버로 전송되지 않습니다.

2. 데이터 저장 방식

  • 모든 시간표 데이터는 앱이 설치된 기기 내부(로컬)에만 저장됩니다.
  • 인터넷 연결 없이 완전히 오프라인으로 동작합니다.
  • 앱을 삭제하면 모든 데이터가 기기에서 완전히 삭제됩니다.

3. 제3자 공유

앱은 사용자 데이터를 제3자에게 공유, 판매, 또는 전송하지 않습니다.

4. 공휴일 데이터

공휴일 연동 기능은 앱에 내장된 공휴일 목록을 사용합니다. 이 과정에서 외부 서버에 요청하거나 사용자 정보를 전송하지 않습니다.

5. 알림 권한

로컬 알림 기능을 사용하기 위해 기기 알림 권한을 요청할 수 있습니다. 알림은 기기 내부에서 처리되며 외부 서버를 사용하지 않습니다.

6. 아동 개인정보 보호

Weeky는 13세 미만 아동으로부터 어떠한 개인정보도 의도적으로 수집하지 않습니다.

7. 방침 변경

본 개인정보처리방침은 변경될 수 있으며, 변경 시 이 페이지에 최신 내용이 반영됩니다.
